📖 IRONWEB — 사용 매뉴얼

🗺️ 배경지도

좌측 패널 최상단에서 4종의 배경지도를 선택할 수 있습니다.

  • OSM 표준 (한국어) — 한글 지명이 포함된 기본 지도
  • 실루엣 (최소화) — 인프라 가시성을 높인 최소 배경 + 행정구역 경계선
  • 다크 — 어두운 톤, 야간 프레젠테이션에 적합
  • 위성 — ArcGIS 위성 이미지

⚡ 송전선로

765kV(빨강), 345kV(주황), 154kV(골드) 3단계로 표시됩니다. (색상 커스텀 가능)

체크박스로 전압별 개별 ON/OFF 가능. 검은 casing이 적용되어 어떤 배경에서도 선명합니다.

🏗️ 시설물

  • 변전소 (보라색) — 전국 변전소 위치. 줌인 시 이름 라벨 표시 (색상 커스텀 가능)
  • 발전소 (전체) (파란색) — 모든 발전소를 통합 표시 (색상 커스텀 가능)

"발전소 전체"와 "연료별" 레이어는 독립적으로 동작합니다. 전체를 끄고 연료별만 볼 수도 있습니다.

🔥 발전소 연료별

발전소를 연료원별로 색상 구분하여 표시합니다. (색상 커스텀 가능)

  • 🟠 LNG/가스 — 주황
  • 🟤 석탄/화력 — 갈색
  • 💚 원자력 — 형광 녹색
  • 🟡 태양광 — 골드
  • 🔷 풍력 — 시안
  • 🔵 수력 — 파랑
  • ⚪ 기타/미분류 — 회색

각 연료 옆에 해당 연료의 발전소 수가 표시됩니다.

🗾 행정구역 (시·도)

17개 시·도를 체크박스로 개별 하이라이트할 수 있습니다.

선택한 지역은 밝게, 나머지는 어둡게 마스킹됩니다.

🏙️ 주요 도시

용인, 청주, 평택, 이천, 파주, 울산, 광양, 포항, 보령 등 LNG 사업 관련 주요 도시의 경계를 하이라이트합니다.

행정구역과 동일하게 마스크 hole로 적용됩니다.

🌓 지도 가리기

  • 반투명 — 선택하지 않은 지역을 반투명 검정으로 덮어 주변 맥락을 유지
  • 불투명 — 완전 검정으로 가려 프레젠테이션/캡쳐에 적합

📷 캡쳐

  • 📷 전체 캡쳐 — 현재 화면 전체를 PNG로 저장
  • ✂️ 영역 캡쳐 — 드래그로 원하는 영역만 선택하여 PNG 저장

💾 오프라인 HTML 내보내기

현재 로드된 모든 데이터를 HTML 파일 안에 내장하여 저장합니다.

사용법: 인터넷 환경에서 데이터 로드 완료 → "💾 오프라인 HTML 내보내기" 클릭 → 다운로드된 HTML을 VDI에 복사하여 사용

오프라인 파일은 네트워크 없이 즉시 모든 인프라 데이터를 표시합니다. (배경지도 타일만 인터넷 필요)

🔍 클릭 상세 정보

지도 위 시설물(송전선로, 변전소, 발전소)을 클릭하면 우측에 상세 정보 패널이 표시됩니다.

전압, 연료 타입, 운영사 등이 배지와 속성 테이블로 표시됩니다.

🎨 색상 커스텀

각 레이어 이름 옆의 컬러 피커(■)를 클릭하면 색상을 자유롭게 변경할 수 있습니다.

  • 송전선로 (765kV, 345kV, 154kV) — 각 전압별 선 색상
  • 변전소, 발전소 (전체) — 점 색상
  • 연료별 발전소 (LNG, 석탄, 원자력 등 7종) — 각각 독립 색상

변경된 색상은 브라우저에 자동 저장되어 다음 접속 시에도 유지됩니다. 초기화하려면 브라우저의 localStorage를 삭제하거나 새 시크릿 창에서 열면 됩니다.

🏗️ 아이소메트릭 아이콘

줌 레벨 8 이상에서 변전소와 발전소가 원형 점 대신 미니어처 아이콘으로 표시됩니다.

  • 변전소 — 변압기 박스 + 절연체
  • LNG/가스 — 가스터빈 + 탱크
  • 석탄/화력 — 공장 + 석탄더미
  • 원자력 — 돔 원자로
  • 태양광 — 기울어진 패널
  • 풍력 — 터빈
  • 수력 — 댐

3D 기울기(◇ 버튼 또는 3 키) 시 아이콘이 지면에 눕어서 입체적으로 보입니다.

⌨️ 단축키

  • D — 다운로드 상태창 열기/닫기
  • ESC — 정보 패널 닫기 / 영역 캡쳐 취소 / 매뉴얼 닫기
  • 3 — 3D 기울기 토글

📊 통계

좌측 패널 하단 "통계" 섹션에서 전압별 선로 수, 연료별 발전소 수를 확인할 수 있습니다.

🔌 오프라인/VDI 사용

사내망/VDI에서 외부 CDN이 차단될 경우를 대비한 구성:

  • setup.sh (Mac) 또는 setup.bat (Windows)를 인터넷 환경에서 실행
  • lib/, data/ 폴더에 MapLibre와 행정구역 데이터가 로컬 저장됨
  • ironweb/ 폴더 전체를 VDI로 복사하면 CDN 없이 작동
  • 타일 서버 차단 시 자동으로 오프라인 모드 전환 (배경 없음, 인프라 정상)
